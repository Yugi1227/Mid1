# Experiment 03: Continuous Testing

## Aim
Demonstrate continuous testing using a small calculator program and automated tests.

## Run

```powershell
.\run.ps1
```

## What It Does
- Compiles a Java calculator class.
- Compiles a test class.
- Runs the test class.
- Prints a pass message when all tests succeed.

## Expected Result
The terminal displays `All calculator tests passed`.
