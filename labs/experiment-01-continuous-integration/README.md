# Experiment 01: Continuous Integration with Version Control

## Aim
Implement a simple Continuous Integration workflow using Git and GitHub Actions.

## Run

```powershell
.\run.ps1
```

## What It Does
- Creates a simple Python file.
- Creates a matching test file.
- Adds a GitHub Actions workflow file.
- Runs the test locally.
- Initializes a local Git repository and commits the files.

## Expected Result
The local test passes and Git shows one commit containing the CI files.
