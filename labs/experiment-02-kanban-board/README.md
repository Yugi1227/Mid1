# Experiment 02: Scrum or Kanban Board

## Aim
Create and use a simple Kanban board for task management.

## Run

```powershell
.\run.ps1
```

## What It Does
- Creates a board with four columns.
- Prints the board in the terminal.
- Moves one task from `To Do` to `In Progress`.
- Saves the updated board to `_run\kanban-board.json`.

## Expected Result
The terminal shows the board before and after the task movement.
