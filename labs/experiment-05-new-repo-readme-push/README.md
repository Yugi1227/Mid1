# Experiment 05: Create Repository, Add README, and Push

## Aim
Create a new Git repository, add a README file, commit it, and push it to a remote repository.

## Run

```powershell
.\run.ps1
```

## What It Does
- Creates a local repository.
- Creates `README.md`.
- Commits the file.
- Pushes to a local bare remote so the experiment works without GitHub login.

## Expected Result
The local repository pushes successfully to the demo remote.
