# Experiment 07: View Git Log and Commit Hash

## Aim
Use `git log` to view commit history and find a commit hash.

## Run

```powershell
.\run.ps1
```

## What It Does
- Creates a demo repository.
- Makes three commits.
- Prints the commit history.
- Prints the latest commit hash.

## Expected Result
The terminal displays short commit hashes and the full latest commit hash.
