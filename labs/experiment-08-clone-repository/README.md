# Experiment 08: Clone a Git Repository

## Aim
Clone a Git repository from a remote source.

## Run

```powershell
.\run.ps1
```

## What It Does
- Creates a small source repository.
- Creates a local bare remote.
- Clones the remote repository.
- Shows cloned files and remote URL.

## Expected Result
The repository is cloned into `ClonedRepo` inside the `_run` folder.
