# Labs: Experiments 9 to 17 (Ready to Run)

This folder contains ready-to-run files and commands for Experiments 9 to 17.

## Quick setup (Windows PowerShell)

1. Install Selenium once:

   pip install selenium

2. Keep Edge WebDriver in this workspace at:

   C:\Users\yuges\Downloads\edgedriver_win64\msedgedriver.exe

3. For Docker experiments, make sure Docker Desktop is running.

4. Use the experiment-specific folders below.
