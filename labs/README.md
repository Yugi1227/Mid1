# DevOps Labs: Experiments 01 to 17

This folder follows the DevOps manual and includes runnable examples for experiments 01 to 17.

## How to Run

Open any experiment folder and run:

```powershell
.\run.ps1
```

## Requirements

- Git for experiments 01 and 04 to 09.
- Python for experiment 01.
- Java JDK for experiment 03.
- Python, Selenium, Microsoft Edge, and `msedgedriver.exe` for experiments 10 to 12.
- Docker Desktop for experiments 13 to 17.

For Selenium, keep `msedgedriver.exe` in the project root or set `EDGE_DRIVER_PATH`.
